## ⌨️ COMO APLICAR O TÍTULO (Tutorial)

Existem duas formas de colocar o título no seu amigo. Para isso, utilizamos o comando `/nickname` do mod Essential Commands.

### Opção A: Pelo Terminal (Console)
Se você estiver fora do jogo ou pelo shell do Linux:
1. Acesse o console interativo usando o atalho rápido: `mcconsole`
2. Digite o comando (sem a barra): `nickname set <NomeDoAmigo> <Título_Desejado>`

**Exemplo Real:**
`nickname set Vinicius O_Lenda_do_Reino`

*(Aperte Ctrl+A e depois D para sair do mcconsole)*

### Opção B: Dentro do Jogo
Se você for operador (`/op <SeuNome>` no console):
1. Aperte `T` para abrir o chat.
2. Digite: `/nickname set <NomeDoAmigo> <Título>`

---

## 💡 DICAS IMPORTANTES:
- **Espaços:** Se o título que você quer colocar tiver espaços (ex: "O Rico do Vale"), use aspas para o comando entender ou _underlines_ para aplicar sem erro: 
  👉 `/nickname set Vinicius "O Lenda do Reino"`
- **Placeholder:** O comando `/nickname` pertence ao mod **Essential Commands**. Após definido, esse título substituirá o nome do jogador no chat graças à integração com o mod `Styled Chat` e `PlaceholderAPI` que também estão no servidor.
- **Limpar/Alterar:** Se o "cria" mudar de cargo e perder a utilidade original, você pode retirá-lo digitando `/nickname clear <NomeDoAmigo>`.
